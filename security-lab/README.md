# Security Lab

Practical defensive security notes and mini-labs for SOC L1 / Linux Support preparation.

## Contents

- `linux-auth-log-analysis.md` - Linux authentication log analysis
- `ssh-failed-login-triage.md` - SSH failed login triage workflow
- `network-traffic-triage.md` - basic network traffic checks with tcpdump/ss/dig/curl
- `incident-report-template.md` - basic incident report template
- `scripts/auth-summary.sh` - small helper script for auth log review

## Focus

- log analysis
- SSH authentication events
- Linux service checks
- network basics
- evidence collection
- false positive vs true positive thinking
- simple incident documentation

## Safe scope

This lab is defensive only. It focuses on logs, monitoring, troubleshooting and incident triage.
